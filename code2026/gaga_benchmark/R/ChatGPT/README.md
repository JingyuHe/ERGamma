# ChatGPT GaGa Benchmark Notes

This folder keeps two separate benchmark paths.

`quad_gaga_pp()` is the older adaptive quadrature path.  It is labelled
`Quad_GaGa`; it is not PIG-GaGa.  Old outputs named `*_pig_gaga_*` were
generated by the mislabeled quadrature pipeline and have been moved to
`results/deprecated_mislabeled_pig/`.

The actual PIG data-augmentation benchmark is the MCMC path:

```text
PIG_GAGA_MCMC_FORMULAS.md       mathematical specification
pig_mcmc_gaga_lib.R             per-gene PIG-augmented MCMC sampler
04_armstrong_pig_mcmc_gaga.R    Armstrong vs gaga::fitGG
05_maqc_pig_mcmc_gaga.R         MAQC vs gaga::fitGG and limma_F
06_run_pig_mcmc_all.R           run both real-data examples
07_simulate_stirling_vs_pig_mcmc.R
                                simulation: same fitGG EM hyperparameters,
                                Stirling scorer vs PIG-MCMC scorer
08_oracle_stirling_pig_simulation.R
                                oracle simulation with exact quadrature,
                                Stirling scorer, and PIG-MCMC scorer
09_summarize_oracle_paper_simulation.R
                                combines per-scenario oracle simulation runs
10_empirical_report.R           organizes original GaGa empirical reproduction
                                and PIG-MCMC empirical results
```

The MCMC path samples `z_i` and `alpha_i`; benchmark posterior probabilities
use the Rao-Blackwellized average of `Pr(z_i | alpha_i, x_i)` over retained
draws.  Each `alpha_i` update calls `pig_rPIG_original()` and then samples
`log(alpha_i)` from the PIG-augmented conditional by stepping-out slice
sampling, as documented in `PIG_GAGA_MCMC_FORMULAS.md`.

The PIG draw in `pig_mcmc_gaga_lib.R` is copied into
`pig_rPIG_original()` from the root `sampling.R` / `code2026/sampling.R`
parameterization: finite `GIG(-3/2, chi=1/(2k^2), psi=2c^2)` terms plus the
Gamma tail correction.  It intentionally does not use the older
`Jianeng_Curriculum/sampling.R` convention, which passes `sqrt(2) * alpha` and
omits the tail correction.

Smoke-test run:

```sh
PIG_MCMC_ARM_MAX_GENES=80 PIG_MCMC_ITER=80 PIG_MCMC_BURNIN=30 \
  Rscript code2026/gaga_benchmark/R/ChatGPT/04_armstrong_pig_mcmc_gaga.R

PIG_MCMC_MAQC_MAX_PROBES=80 PIG_MCMC_ITER=60 PIG_MCMC_BURNIN=20 \
  Rscript code2026/gaga_benchmark/R/ChatGPT/05_maqc_pig_mcmc_gaga.R
```
