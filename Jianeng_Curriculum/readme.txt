Code of Bayesian Inference for Polya Inverse Gamma Models

1. sampling.R is the main function of Gibbs sampler, MH sampler used in all other simulation files. 

2. gamma_shape.R generates plots and tables in section 4, inference of gamma shape

3. simulation_study.R generates plots and tables in section 3, dirichlet and multinomial models

4. Opioid.R is the code of section 5, empirical illustration of opioid and prescription drug data
